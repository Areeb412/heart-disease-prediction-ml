# Heart Disease Prediction Machine Learning Project

## Overview

This project implements a comprehensive machine learning pipeline for predicting heart disease using the UCI Heart Disease dataset. It demonstrates end-to-end data science skills including data preprocessing, exploratory data analysis, feature engineering, model training, and performance evaluation.

## Key Features

- **Data Exploration**: Comprehensive analysis of patient demographics and medical indicators
- **Feature Engineering**: Handling categorical variables, scaling, and feature selection
- **Model Development**: Implementation and comparison of Logistic Regression and Decision Tree models
- **Performance Evaluation**: Detailed metrics including accuracy, precision, recall, F1-score, and ROC-AUC
- **Medical Insights**: Interpretation of results with domain-specific context

## Technologies and Libraries

- **Python** - Core programming language
- **Pandas & NumPy** - Data manipulation and numerical computing
- **Scikit-learn** - Machine learning algorithms and evaluation metrics
- **Matplotlib & Seaborn** - Data visualization and plotting
- **Jupyter Notebook** - Interactive development environment

## Dataset

The project uses the UCI Heart Disease dataset (Cleveland), containing 303 patient records with 13 features including age, sex, chest pain type, resting blood pressure, cholesterol levels, and other cardiac indicators.

## Project Structure

- `heart.ipynb` - Main analysis notebook with complete workflow
- `data/heart.csv` - Dataset file
- `README.md` - Project documentation

## Usage

1. Ensure Python 3.7+ is installed
2. Install required dependencies: `pip install pandas numpy scikit-learn matplotlib seaborn`
3. Open `heart.ipynb` in Jupyter Notebook or JupyterLab
4. Run cells sequentially to reproduce the analysis
